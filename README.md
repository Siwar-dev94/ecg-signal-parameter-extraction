# ECG Signal Parameter Extraction using Machine Learning

##  Project Description

This project focuses on the study and extraction of key parameters from ECG (Electrocardiogram) signals using machine learning and signal processing techniques.

##  Notebook

The full implementation is available in the Jupyter Notebook:
📁 `notebook/ecg_analysis.ipynb`

##  Objectives

* Analyze ECG signals
* Detect peaks (QRS complex)
* Extract important features (heart rate, intervals)
* Apply machine learning techniques

##  Tools & Technologies

* Python
* Google Colab
* NumPy, Pandas
* Matplotlib
* Scikit-learn / Deep Learning

##  Results

The notebook demonstrates effective ECG signal preprocessing, feature extraction, and analysis using ML models.
